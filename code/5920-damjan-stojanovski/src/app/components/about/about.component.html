Project Information

This is a Movie Management System built for the Internet Programming course exam. The application allows users to view, create, edit, and delete movies with detailed information including cast, genres, ratings, and Oscar awards.

Features:

  • Movie listing with sorting and filtering
  • Detailed movie information display
  • Create and edit movies
  • Delete movies with confirmation
  • Statistics and analytics
  • Responsive design for all devices

Technical Details

Frontend Framework: Angular 17
Language: TypeScript
Styling: CSS3 with responsive design
API: RESTful API integration
